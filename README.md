## Getting Started

To run this locally:

1. Clone the repository
   ```bash
   git clone https://github.com/Turzo2134/turzo2134.github.io.git
   cd turzo2134.github.io
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx http-server
   ```
